Coding Exercise 42: Sum Array Exercise

index.js

```javascript
// DEFINE YOUR FUNCTION BELOW:

function sumArray(arr) {
  let sum = 0;
  for (let num of arr) {
    sum += num;
  }
  return sum;
}
```
