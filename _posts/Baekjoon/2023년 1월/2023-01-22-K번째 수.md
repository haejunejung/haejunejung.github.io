# **_BOJ 11004 K번째 수_**

- `정렬` 알고리즘

<hr>

## 풀이

- 오름차순으로 정렬한 후, k번째 수 출력

```c++
sort(A, A + N, [](long long &a, long long &b)
        { return a < b; });

cout << A[k - 1]; // 0~(k-1)로 총 k번째 수
```

<hr>

## 💡 **_Code_**

```c++
#include <iostream>
#include <algorithm>

using namespace std;

const unsigned int MAX = 5000000;

void input()
{
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);
}

long long A[MAX];

int main()
{
    input();

    int N, k;
    cin >> N >> k;

    for (size_t i = 0; i < N; ++i)
    {
        cin >> A[i];
    }

    sort(A, A + N, [](long long &a, long long &b)
         { return a < b; });

    cout << A[k - 1];

    return 0;
}
```
