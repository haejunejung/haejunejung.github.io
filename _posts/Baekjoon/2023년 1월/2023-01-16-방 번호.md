# **_BOJ 1475 방 번호_**

## 풀이

- 0~9까지의 숫자세트를 받아낼 수 있는 배열 선언
- 6과 9는 뒤집어서 사용할 수 있다 → 6 또는 9가 나왔을 때, 더 적은 곳에 개수를 추가한다

```c++
int val = str[i] - '0';
if (val == 6 || val == 9)
{
    if (count[6] > count[9])
        count[9]++;
    else
        count[6]++;
}
```

- string으로 입력을 받은 것이 아니라, getline을 통해 입력을 받았기 때문에 `'\0'`에서 종료한다

```c++
if (str[i] == '\0')
    break;
```

## 💡 **_Code_**

```c++
#include <iostream>

using namespace std;

const int Max = 1000000;

void input()
{
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    cout.tie(nullptr);
}

int main()
{
    input();

    char str[Max];
    cin.getline(str, Max, '\n');

    int count[10] = {};
    for (size_t i = 0; i < Max; ++i)
    {
        if (str[i] == '\0')
            break;
        int val = str[i] - '0';
        if (val == 6 || val == 9)
        {
            if (count[6] > count[9])
                count[9]++;
            else
                count[6]++;
        }
        else
            count[val]++;
    }

    int res = 0;
    for (size_t i = 0; i < 10; ++i)
    {
        res = max(res, count[i]);
    }

    cout << res;

    return 0;
}
```
